🚀 How It Works

✅ Step 1: Run the Project

Run the main class to start the Spring Boot application:

src/main/java/JFS6WDE/OnlineBusTicketBooking/OnlineBusTicketBookingApplication.java

🔐 Step 2: Access the Admin Panel

Login using these admin credentials:

  • Email: admin@admin.com
  • Password: admin

After logging in, you can manage bus details from the dashboard.

Default admin credentials can be changed at:

Services/UserServiceImpl.java (Line 54)

👤 Step 3: User Registration & Booking

Register with your email and password. After login, you'll land on the user homepage.

  • View available buses
  • Book tickets
  • Search buses
  • View booking history
  • Cancel bookings
