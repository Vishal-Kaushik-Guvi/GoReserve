🚌 GoReserve - Online Bus Ticket Booking System Hello and welcome to the GoReserve Web Project! I'm Vishal Kaushik from Sonipat, Haryana, and I'm excited to share this project with you.

This is a fully functional Bus Booking Web Application built using modern technologies for both frontend and backend development.

🛠️ Tech Stack

Frontend:

HTML

CSS

JavaScript

Tailwind CSS

Bootstrap

Backend:

Java (Spring Boot, JDK 21)

Maven

Database:

MySQL (localhost:3306) (You can switch to any database of your choice through application.properties Path -(OnlineBusTicketBooking\src\main\resources\application.properties))

🚀 How It Works ✅ Step 1: Run the Project Start the application by running the main class: src/main/java/JFS6WDE/OnlineBusTicketBooking/OnlineBusTicketBookingApplication.java

🔐 Step 2: Access the Admin Panel Register using the admin credentials:

Email: admin@admin.com

Password: admin

After logging in, you'll be redirected to the Admin Dashboard.

Here, you can:

Add new buses

Update existing bus details

Delete buses from the system

Note: You can change the default admin email and password in: UserServiceImpl.java at line 54 (Path: JFS6WDE\OnlineBusTicketBooking\Services\UserServiceImpl.java)

👤 Step 3: User Registration & Booking Register as a normal user with any email and password.

Once logged in, you'll land on the User Home Page (Index).

From here, you can:

View available buses

Book tickets

Search Buses

View your booking history

Cancel bookings if needed
